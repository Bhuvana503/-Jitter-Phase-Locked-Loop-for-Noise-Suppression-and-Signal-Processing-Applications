# -Jitter-Phase-Locked-Loop-for-Noise-Suppression-and-Signal-Processing-Applications
🚀 Low-jitter charge-pump PLL frequency synthesizer designed and verified in LTspice for noise suppression and precision signal processing.

This project was completed as part of the IEEE CAS SIP internship under the Digital Schematics domain, focusing on jitter-aware PLL design for noise-immune clock generation and signal-processing systems.

This project presents the design and simulation of a complete Phase-Locked Loop (PLL) frequency synthesizer using LTspice.

The system is built for low-jitter clock generation, noise suppression, and precision signal-processing applications.

The architecture includes:

• D flip-flop based Phase Frequency Detector (PFD) with tri-state reset logic.

• Symmetric PMOS/NMOS charge pump.

• Second-order passive loop filter.

• Voltage-Controlled Oscillator (VCO).

• Edge detection and programmable feedback divider.

• The PLL demonstrates closed-loop frequency synthesis (10 MHz reference to 100 MHz output), stable lock behavior, near-zero steady-state phase error, and clean spectral output verified through transient and FFT analysis.

• This work reflects practical mixed-signal design methodology and verification flow used in analog/digital schematic development.
